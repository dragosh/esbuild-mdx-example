# esbuild-mdx-example

Reference implementation for [esbuild-mdx](https://github.com/zaydek/esbuild-mdx).

To use this project:

```sh
git clone https://github.com/zaydek/esbuild-mdx-example.git
cd esbuild-mdx-example
yarn
yarn build
open out/index.html
```
